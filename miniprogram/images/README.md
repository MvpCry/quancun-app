# TabBar 图标说明

请在此目录下放置以下8个图标文件（PNG格式，建议 81x81 像素）：

1. tab-home.png          - 首页图标(未选中)
2. tab-home-active.png   - 首页图标(选中)
3. tab-attraction.png    - 景点图标(未选中)
4. tab-attraction-active.png - 景点图标(选中)
5. tab-route.png         - 路线图标(未选中)
6. tab-route-active.png  - 路线图标(选中)
7. tab-mine.png          - 我的图标(未选中)
8. tab-mine-active.png   - 我的图标(选中)

另外还需要：
- marker-attraction.png  - 地图景点标记图标
- default-avatar.png     - 默认用户头像

图标资源可以从以下渠道获取：
- 阿里巴巴矢量图标库 (iconfont.cn)
- 微信官方设计资源
- 自行设计
